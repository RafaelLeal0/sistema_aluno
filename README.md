# 🎓 Sistema de Gestão de Alunos
Sistema para cadastro e gerenciamento de alunos com interface gráfica.

# ✨ Funcionalidades
✅ Cadastro de alunos com validação

📊 Tabela para visualização dos dados

🔍 Filtro por nota mínima

💾 Salvar e carregar dados em CSV

📈 Exportar relatórios filtrados

🚀 Como Usar
Instalar dependência:

bash
pip install pandas
Executar sistema:

bash
python main.py
# 📋 Uso Básico
Cadastrar: Preencha os campos e clique em "Cadastrar Aluno"

Filtrar: Digite a nota mínima e clique em "Aplicar Filtro"

Salvar: Clique em "Salvar CSV" para guardar os dados

Exportar: Clique em "Exportar Relatório" para gerar CSV filtrado

🛠 Tecnologias
Python

Tkinter (Interface)

Pandas (Dados)

CSV (Armazenamento)
